# Steep sample

## Requirements

- Ruby 2.5.3

## Install

```
bundle
```

## Execute

### Type check

```
bundle exec steep check execute.rb
bundle exec steep check execute_invalid.rb
bundle exec steep check average.rb
bundle exec steep check average_invalid.rb
```

### Execute Ruby

```
ruby execute.rb
ruby execute_invalid.rb
ruby average.rb
ruby average_invalid.rb
```
