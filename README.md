# Steep sample

## Requirements

- Ruby 2.5.3

## Install

```
bundle
bundle binstubs steep
```

## Execute

### Type check

```
bin/steep check execute.rb
bin/steep check execute_invalid.rb
bin/steep check average.rb
bin/steep check average_invalid.rb
```

### Execute Ruby

```
ruby execute.rb
ruby execute_invalid.rb
ruby average.rb
ruby average_invalid.rb
```
